About - UWI Course Recommender System
This system is an intelligent course recommender system that will be used by the registered students. The course recommender system is a web-based application. It will focus on a student's course field including major(s) and/or minor(s) when making course recommendations. The system will also be able to make recommendations based on certain interests that students have chosen, relativity to their major and minor, and feedback from past students. Ideally, the recommender system should allow users to find the preferable courses quickly and to alleviate the information overload problem. The course recommender system will be a web based application that will be developed using Flask, a python web application framework and AngularJS, a JavaScript-based web application framework. The system will also be using a database; to store all relevant information needed to make recommendations to students. The system will recommend suitable courses to individual students, taking into consideration any previous courses done by the student which may be needed as prerequisites for other courses.
