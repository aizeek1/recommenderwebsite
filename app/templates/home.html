How does it work?



The UWI Course Recommender System recommends courses based on:

Major/Minor

Recommends courses based on the student's major/minor from the database, where related courses have not yet been completed, and prerequisites have been satisfied.

Foundation

The system will obtain the student's faculty, and displays the appropriate courses accordingly.

Interest

This tool allows student to find the best free electives bases on their selected interest and the prerequisites they have completed.

Ratings

Ratings recommendation tool using sentient analysis to find the courses that other students have completed and rated using collaborative filtering

Search

Students are able to search for one or more courses by specifying the coursecode.

Grades

Grades is an in-built function that predictes the probability of getting a certain grade on selective courses based on academic research results
